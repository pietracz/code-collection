AWS Direct Connect (DX)
On Premise Netzwerk wird über eine VPN Verbindung in einem privaten Netzwerk zur VPC erstellt
Es wird geschaut, wie du an deinem Standort schnellstmöglich ins AWS Netzwerk, über eine private Kabelverbindung, einsteigen kannst.
Das KANN auch über Edge Locations laufen, um schneller ins AWS Netz zu kommen.
Reservierung und Einrichtung der dedizierten Leitung kann bis zu 30 Tage dauern.

Allgemein ist bei einem VPN Tunnel ein Customer Gateway auf AWS Kunden Seite notwendig (Eingang) und das Virtual Private Gateway (Ausgang) in der AWS Cloud
